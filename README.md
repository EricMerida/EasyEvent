EasyEvent

A full-stack event management application where users can create events, browse available events, and book or cancel event reservations.
Built with React, Node.js, Express, GraphQL, and MongoDB.

//Features:

User authentication (signup/login)
Create events
Browse available events
View event details
Book events
Cancel bookings
View all personal bookings
Event ownership detection
Responsive event card UI
GraphQL API
DataLoader optimization for database queries
Modal UI for event and booking details
Loading spinner for async operations

//Tech Stack
Frontend:
React
React Router
CSS
Fetch API

Backend:
Node.js
Express
GraphQL
DataLoader
Database
MongoDB
Mongoose
